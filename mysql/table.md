# Table
* create table

```sql
CREATE TABLE individual ( 
	firstname  	VARCHAR(20) NOT NULL,
	lastname   	VARCHAR(20) NOT NULL,
	birthdate  	DATETIME,
	gender     	VARCHAR(1),
	initial    	NUMBER(2),
	farorite   	VARCHAR(6), 
	type       	VARCHAR(8)
);
```

* drop table

``` sql
	DROP TABLE table_name;
```
